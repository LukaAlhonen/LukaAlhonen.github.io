Cash Register System

A cash register system I created together with my group for a software development course.
The app is written in Java and was created according to specifications.
The cashbox, cardreader and product catalogs are emulated by Java applications.
A user can "scan an item" by typing in the barcode associated with the item and remove an item by selecting it from the UI and clicking the remove button
A purchase can be made in cash or with a card payment and works by sending a POST request to the cashbox or cardreader.
